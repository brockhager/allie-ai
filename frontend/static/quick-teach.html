Quick Teach

Use this page to quickly teach Allie a correction, add facts to memory, or mark a conversation response as wrong.

Quick Teach Tips

  • Provide a short, accurate correction to help the model learn.
  • If you know the conversation id and message index, "Mark as Wrong" will associate the report with that conversation.
  • "Request Better Answer" will ask Allie to regenerate a better reply for the flagged message (only if conv id & index provided).
